关于Integer的一些说明

Veröffentlicht am 2017-03-29

Integer

声明一个int类型主要有一下几种方法：

1.Integer a = 66;

这种方法编译器会默认调用Integer.valueOf()静态方法

同时每个Integer类捏弄有一个IntegerCache，里面存放事先创建的-128-127的Integer实例

2.Integer b = new Integer(66);

直接调用构造器创建新对象

3.Int e = 10;

当使用final修饰的时候，直接在运行时常量池中添加10
